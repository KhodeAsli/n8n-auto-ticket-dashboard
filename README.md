## 📋 About This Application

n8n-auto-ticket-dashboard is a modern web application designed for IT helpdesks. It helps teams manage IT support tickets with clear dashboards showing real-time information. You can track staff work, view analytics on ticket trends, and spot unusual issues using built-in smart detection. The dashboard runs smoothly on Windows and offers a clean, responsive design that works well on desktops and laptops.

The app uses the latest web technologies like Next.js, TypeScript, Tailwind CSS, and SQL Server. It also integrates with n8n automation for streamlined task flows, though you don't need to set this up yourself to use the basic features.
